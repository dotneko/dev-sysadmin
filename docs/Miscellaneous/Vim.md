# Simple .VIMRC file

```
set number
set showmatch
set visualbell
set hlsearch
set smartcase
set ignorecase
set incsearch
set autoindent
set expandtab
set shiftwidth=4
set smartindent
set softtabstop=4
set ruler
set backspace=indent,eol,start
```
